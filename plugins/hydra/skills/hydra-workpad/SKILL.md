# Hydra Workpad

Use this skill when updating Linear progress from a Hydra-managed run.

Maintain exactly one persistent Linear comment headed `## Codex Workpad`. Keep it truthful and current with:

- Environment stamp
- Plan
- Acceptance criteria
- Notes
- Validation
- PR state
- Blockers

Do not check off PR, review, or state-transition items until they are actually complete.
